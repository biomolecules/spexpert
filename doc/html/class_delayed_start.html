Detailed Description

This class is helper class, which wraps the WaitTask object if it has some WaitTask::initialDelay() in the WaitWorker::delayedAddWaitTask() method. Its method is invoked by a QTimer after the initialy delay elapses. The delayedAddWaitTask() emits addTask() signal, which is connected to the WaitWorker::addDelayedWaitTask() method and then it destroyes itself.
