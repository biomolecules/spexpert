Detailed Description

This class contains the WaitTask in the WaitTaskListTraits::TaskItem, which will be submited to the list of tasks which will be waited for, when added to the WaitTaskList. Each WaitTask should be handled by some WaitingTask to have some effect, so there shouldn't be any WaitExpTask added to the WaitTaskList without WaitingTask added somewhere after it.
